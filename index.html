🔧 Логические операторы в JavaScript

📚 Теория

&& (И)

Возвращает true, если все операнды истинны

true && true → true
true && false → false
false && true → false

|| (ИЛИ)

Возвращает true, если хотя бы один операнд истинный

true || false → true
false || true → true
false || false → false

! (НЕ)

Преобразует в булево и возвращает противоположное

!true → false
!false → true
!"" → true
!"hello" → false

⚡ Short-circuit evaluation

Вычисление прекращается, как только результат становится известен:

false && anything → false (anything не вычисляется)
true || anything → true (anything не вычисляется)
// Возвращается значение последнего вычисленного операнда

❌ Ложные значения (falsy):

  • false
  • 0
  • "" (пустая строка)
  • null
  • undefined
  • NaN

✅ Все остальные значения - истинные (truthy)
